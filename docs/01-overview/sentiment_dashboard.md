## 组件状态

### 统计卡片状态

| 状态 | 样式 |
|------|------|
| 默认 | bg: #1E293B, shadow: card |
| 悬浮 | translate: -2px, shadow: hover |
| 加载 | skeleton 骨架屏 |

### 风险卡片状态

| 状态 | 样式 |
|------|------|
| 有风险 | 红色边框，警告图标 |
| 无风险 | 灰色背景，显示"暂无风险" |
| 加载中 | skeleton 骨架屏 |

---

## 交互动效

1. **卡片入场**: 依次从上到下渐入，间隔 100ms
2. **数字跳动**: CountUp 动画，1s 完成
3. **圆环图**: 从 0° 顺时针展开，1s
4. **悬浮提示**: fade + scale，0.2s
