## 功能测试

### 1. 股票注册
```bash
POST /api/v1/stocks
{
  "code": "601872.SH",
  "name": "招商轮船"
}

Response:
{
  "code": "601872.SH",
  "name": "招商轮船",
  "keywords": null,
  "id": 1,
  "created_at": "2026-03-01T18:00:52",
  "updated_at": null
}
```
✅ **通过**

### 2. 新闻抓取
```bash
POST /api/v1/news/fetch
{
  "limit": 10
}

Response:
{
  "fetched_count": 10,
  "saved_count": 10,
  "duplicates": 0,
  "message": "成功抓取 10 条新闻，保存 10 条新记录，跳过 0 条重复"
}
```
✅ **通过** - 成功抓取 10 条招商轮船新闻

### 3. 新闻去重
```bash
第二次抓取（同样参数）

Response:
{
  "fetched_count": 10,
  "saved_count": 0,
  "duplicates": 10,
  "message": "成功抓取 10 条新闻，保存 0 条新记录，跳过 10 条重复"
}
```
✅ **通过** - 去重功能正常

### 4. 获取新闻列表
```bash
GET /api/v1/news?limit=5

Response:
{
  "total": 10,
  "news": [...]
}
```
✅ **通过** - 返回 10 条新闻，按时间倒序

### 5. 获取指定股票新闻
```bash
GET /api/v1/news/1?limit=5

Response:
{
  "total": 10,
  "news": [...]
}
```
✅ **通过** - 返回 stock_id=1 的所有新闻
